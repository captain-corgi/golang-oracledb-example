# Golang Oracle DB Example

This is a small example of connect and use Oracle DB in Golang

> Container Database:
> 
> Username: SYS
> Role: SYSDBA
> Password: Oradoc_db1
> Host: localhost
> SID: ORCLCDB
> 
> Pluggable Database:
> 
> Username: SYS
> Role: SYSDBA
> Password: Oradoc_db1
> Host: localhost
> Service Name: ORCLPDB1.localdomain
>
